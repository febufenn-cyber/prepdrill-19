# Phase 13 Blind Spots

- Multiple models can agree on the same wrong or ambiguous answer.
- A structurally valid variant can drift away from the intended syllabus concept.
- Near duplicates can inflate coverage while leaking source wording.
- Difficulty labels generated by a model are hypotheses, not evidence.
- Human approval must preserve generated provenance rather than relabeling the item as official.
- A promoted experimental item should still be excluded from official archives, full mocks, and mastery unless separate policy explicitly enables it.
- Shadow performance can look good while complaints reveal ambiguity or cultural/linguistic issues.
- Stop rules must be automatic; relying on someone noticing a dashboard is unsafe.
- Generation economics must include rejected candidates and reviewer time, not only model-call cost.
