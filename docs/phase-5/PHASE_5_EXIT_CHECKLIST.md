- [x] Grounding bundles pin exact revision, answer, concept, and evidence.
- [x] Structured explanation layers are required.
- [x] Claimed answers must match canonical evidence.
- [x] Evidence references must belong to the grounding bundle.
- [x] Selected-distractor analysis is required.
- [x] Prompt/model/revision-sensitive deterministic cache.
- [x] Model routing and hard cost ceiling.
- [x] Provider outage returns explicit unavailable state.
- [x] Contradictions enter an immutable review queue.
- [x] Only approved or unavailable responses are learner-visible.
- [x] AI has no canonical answer, publication, score, or mastery write path.
- [x] Evaluator and inherited workflows pass.

## Real operations
- [ ] Production provider credentials configured.
- [ ] Golden cross-format explanation set reviewed by subject experts.
- [ ] Cost and latency budgets measured in production.
- [ ] Reviewer staffing and SLA approved.
- [ ] No high-severity contradiction in the release set.

Software completion does not claim expert approval or live-provider readiness.
